  • What is Linux?
    • Linux is an Operating System
    • Operating Systems are how we can communicate with the hardware of a device
    • A Brief History of Linux
    • UNIX Introduction
  • Common Uses for Linux
    • Phones and Tablets (Android is Linux Based) and about 70% of the market as of 2022
    • Servers, over 80% of web sites run some version of Linux/Unix
    • Supercomputers, starting in November of 2017, all 500 of the top Supercomputers run some version of Linux
  • Advantages and Disadvantages of Linux
    • Advantages
      • Why Linux has better Security
      • Almost all versions of Linux are Free!
      • Linux can handle less resources so you can use older computers longer, and cheap computers with less power (ex. Raspberry Pi)
      • Everything can be customized!
      • There are a lot of options to choose from
    • Disadvantages
      • Linux is less popular on Desktop Computers so most people aren't familiar with it, and the learning curve can be rough.
      • Compatibility, because it's not widely adopted, Linux programs aren't common, and most things are made for Windows.
      • There is NO tech support, if you run into an issue it's all on you to solve, hope your GoogleFu is strong.
      • Hardware Compatibility, Device drivers are an issue with Linux, it can be hard to find things that work with your system. If you want a printer you can't just go buy one, you need to find one that's compatible.
      • Everything can be customized!
  • Vocabulary
    • Man pages (Short for Manual)
    • Distros or Flavours are which kind of Linux you're using
    • What is Linux and is it more then just a Kernel
