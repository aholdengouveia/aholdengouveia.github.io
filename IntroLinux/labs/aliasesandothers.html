Aliases

Reference: http://linuxreviews.org/quicktips/alias/

  1. For this part we are going to create and alias that will untar a tarball easily. enumerate
  2. Look up the man file on Aliases, take a screenshot.
  3. Create an alias called untar that will run the command tar -xvf take a screenshot.
  4. Create a tarball of your folders. take a screenshot.
  5. Run untar on that tarball. What happened? take a screenshot.
  6. How would you make this a permanent addition to your bash profile? (Don't make it permanent, just tell me how you would do it)
  7. Pick one command that you use all the time, create and alias for it. Tell me what the command is, why you chose is and what you called your alias. take a screenshot.
  8. The answers to all the questions should be put in a file called Aliases

Environment variables

This part of the lab is about changing your command prompt. We will be using Environmental Variables to do this.

Reference: Bash prompt customization guide and Bash prompt tutorial

https://www.maketecheasier.com/more-useful-and-interesting-bash-prompts/

  1. Run the command echo \$SHELL What happened? take a screenshot.
  2. To do this you need to change the variable PS1 and reset it to your desired prompt. Reset this to your full first name. Take a screenshot to pass in
  3. What does your .bash\_profile or .profile say? take a screenshot.
  4. Go to the man file on Bash. You should be able to find options for your prompt. Try adding in the current time option, any of the 4 choices are fine.
  5. Run the command unset PS1 What happened? How did you fix it? take a screenshot.
  6. Go to the second reference link, read through it and make a command prompt that you think will be most useful to you. What did you do? Take a screenshot.
  7. The answers to all the questions should be put in a file called EnvironmentVariables
