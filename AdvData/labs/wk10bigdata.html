Choose your own tutorial

Pick one of the tutorials listed on this GitHub Repo https://github.com/SkalskiP/courses If you already have some Python skill, you could do one of the tutorials at Kaggle https://www.kaggle.com/learn/intro-to-machine-learning instead.

Please take note, there is a wide variety of options here, at all different skill levels and length. You should pick a free one. If you pick a longer option, you don't need to watch/do everything, just start it and do enough to get an idea of how it would go.
