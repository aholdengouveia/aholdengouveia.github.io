Objectives:

  • The objective of this lab is to introduce students to AWK, a powerful text-processing and pattern-matching language commonly used in Linux administration. Through hands-on exercises, students will learn to extract, filter, and manipulate data from structured text files using AWK's field-oriented processing capabilities, pattern matching, and computational features.

Deliverables

A text document with the awk statements used to get each result. Clearly labeled so I can see which answer goes to which question.

Please include the command, a screenshot that includes the command and a few results showing it works as intended, cite all sources you used, and give a short explanation of how the command works and why.
